### 实现的功能如下

* 启动页；

* 登录注册；

* 主页面
	* 下拉刷新，从`Bmob`获取数据；

	* 更新，将本地数据更新到远程`Bmob`后台（将清除本地数据）；

	* 清除本地所有数据；

* 新建笔记

	* 闹钟提醒功能（定时）；
	
	* 优先级功能（排序）；

	* 分享到微信
